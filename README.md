# SlideEffect
图片滑动拼接效果实现
支持将图片沿任意一条切割线分割成两部分，每部分分别可以选择五种动效：                                                                             
1）	从左到右消失                                                                                                                          
2）	从右到左消失                                                                                                                       
3）	从上到下消失                                                                                                                          
4）	从下到上消失                                                                                                                          
5）	沿切线垂直方向消失                                                                                                                     
被分割的两部分移动过程中不会超出切割区域
